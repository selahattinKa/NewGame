**Proje:** Canavar Zindanları (Monster Dungeons) — 2D Idle RPG + Monster Collection + Dungeon Crawler
**Platform:** Mobile (iOS/Android), Unity 6.3 LTS, URP 2D
**Durum:** MVP tasarımı büyük ölçüde tamamlandı (15/18 sistem GDD'ye sahip, 13'ü approved)

**5 Oyun Sütunu:**
1. Cömert Zindan — cömert loot, oyuncu eli boş dönmez
2. Topla Hepsini — geniş canavar koleksiyonu, her canavar benzersiz
3. Senin Tempon — idle veya aktif oynama seçimi
4. Güç Hisset — sürekli güçlenme fantezisi
5. Hep Bir Şey Var — her girişte yapılacak bir şey var

**Tamamlanan GDD'ler (Approved):** Canavar Veritabanı, Element Sistemi, Ekonomi, Kaydetme/Yükleme, Sağlık/Can, Hasar Hesaplama, Canavar Güçlendirme, Loot/Ödül, Takım Kurma, Düşman AI, Hibrit Savaş, Canavar Toplama/Evrim, Zindan Keşif, Otofarm/Idle

**In Review:** UI Framework (design/gdd/ui-framework.md) — önceki inceleme 2026-06-25, 13 blocker çözüldü

**MVP kapsam dışı:** Arena/PvP, günlük görevler, sezon sistemi, tutorial, 2+ zindan bölgesi

**Why:** GDD'lerin hangi aşamada olduğunu ve sistemler arası bağımlılıkları takip etmek için.
**How to apply:** Yeni tasarım kararlarını mevcut approved GDD'lerle cross-check et. UI Framework review için özellikle Otofarm/Idle ve Hibrit Savaş GDD'lerini referans al.
